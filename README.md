GAMBLEEE!!!1!! sorting algorithm inspired by bogo sort
![image alt](https://github.com/AuroraFreire/gambling-sort/blob/main/gambling_flag.png)

**Ok so, here is what this does:**
1. Randomly chooses 2 items from the array
2. Swaps them with eachother
3. Repeats that process until everything is sorted

Simple, right?

**Time Complexity**
1. Best case: O(n)
2. Average case: O(n ⋅ n!)
3. Worst case: O(∞)

**Space Complexity**
O(n)
